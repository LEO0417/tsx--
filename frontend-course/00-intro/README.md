# 第0课：课程介绍与环境搭建

## 学习目标

- 了解前端开发的基本概念
- 理解前端、后端、全栈的区别
- 搭建开发环境
- 熟悉基本的开发工具

## 什么是前端开发？

前端开发是创建网页或应用程序用户界面的过程。用户直接看到和交互的部分都属于前端。

### 前端技术栈

1. **HTML** - 网页的结构和内容
2. **CSS** - 网页的样式和布局
3. **JavaScript** - 网页的交互和动态功能

### 前端 vs 后端

- **前端**：用户界面、交互逻辑、视觉呈现
- **后端**：服务器、数据库、业务逻辑
- **全栈**：同时掌握前端和后端技术

## 开发环境搭建

### 1. 代码编辑器

推荐使用 **Visual Studio Code**

- 下载地址：<https://code.visualstudio.com/>
- 安装后推荐的插件：
  - Live Server（实时预览）
  - Prettier（代码格式化）
  - ESLint（代码检查）

### 2. 浏览器

推荐使用 **Google Chrome**

- 内置强大的开发者工具
- 支持最新的Web标准

### 3. Node.js 和 npm

- 下载地址：<https://nodejs.org/>
- 选择 LTS 版本
- 验证安装：

  ```bash
  node --version
  npm --version
  ```

### 4. Git（版本控制）

- 下载地址：<https://git-scm.com/>
- 基本配置：

  ```bash
  git config --global user.name "你的名字"
  git config --global user.email "你的邮箱"
  ```

## 创建第一个项目

1. 创建项目文件夹
2. 在 VS Code 中打开文件夹
3. 创建基本文件结构：

   ```text
   my-first-project/
   ├── index.html
   ├── style.css
   └── script.js
   ```
